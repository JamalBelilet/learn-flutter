# ListView & GridView

```dart
ListView(
  children: <Widget>[
    AnyWidget(),
    AnyWidget(),
    AnyWidget(),
  ],
),

GridView.count(
  crossAxisCount: 2,
  children: <Widget>[
    AnyWidget(),
    AnyWidget(),
    AnyWidget(),
  ],
),
```

## ListTile and GridTile

```dart
ListView(
  children: <Widget>[
    AnyWidget(),
    ListTile(
      title: AnyWidget(),
      subtitle: AnyWidget(),
      trailing: AnyWidget(),
      leading: AnyWidget(),
    ),
    AnyWidget(),
    AnyWidget(),
  ],
),

GridView(
  children: <Widget>[
    AnyWidget(),
    Card(
      child: AnyWidget(),
    ),
    AnyWidget(),
  ],
),
```
