# Stack

```dart
Stack(
  alignment: Alignment.bottomRight,
  children: <Widget>[
    AnyWidget(), // back
    AnyWidget(), // middle
    AnyWidget(), // front
  ],
),
```

### Positioned and Align

```dart
Stack(
  alignment: Alignment.bottomRight,
  children: <Widget>[
    AnyWidget(),
    Positioned(
      top: 0.0,
      right: 21,
      child: AnyWidget(),
    ),
    AnyWidget(),
    Align(
      alignment: Alignment.topLeft,
      child: AnyWidget(),
    ),
  ],
),
```
